# <p style="text-align: center;"> **Cryptocurrency Recommender** </p>

## **Description**
<p> A simple webpage that displays buy and price of Bitcoin and Ethereum on Blockchain and Binance.</p>

## **How To Use**
1. Clone this repository
2. Open the project in VS Code
3. Open a new terminal in VS Code
4. Navigate to Frontend/my-app dirctory
5. Type command: npm start
6. Open a another new terminal in VS Code
7. Navigate to Backend/ directory
8. Type command: nodemon app/app.js
9. Now open browser and type localhost:3000 inside browser url
